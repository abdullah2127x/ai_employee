```python
# watchers/filesystem_watcher.py

def on_created(self, event):
    """User drops file in Drop/ folder"""
    
    # 1. Detect new file
    src_path = Path(event.src_path)  # "Drop/invoice.pdf"
    
    # 2. Read file content
    file_content = src_path.read_text()
    file_hash = calculate_hash(src_path)
    
    # 3. Create metadata using CENTRALIZED TEMPLATE
    from utils.task_template import create_file_drop_metadata
    
    metadata_content = create_file_drop_metadata(
        original_name=src_path.name,
        original_path=str(src_path),
        content=file_content,
        file_hash=file_hash,
        size=src_path.stat().st_size,
        priority=determine_priority(src_path),
        timestamp=datetime.now()
    )
    
    # 4. Save to Needs_Action/ folder
    metadata_path = self.needs_action / f"FILE_{timestamp}_{src_path.name}.md"
    metadata_path.write_text(metadata_content)
    
    # 5. Move original file to Drop_History/
    shutil.move(str(src_path), str(self.drop_history / src_path.name))
```

### **Phase 3: Claude Runner Processes the File**

```python
# claude_runner.py

def process_task(task_file: Path):
    """Process a single task file"""
    
    # 1. Read task file (metadata + content)
    task_content = task_file.read_text(encoding='utf-8')
    
    # 2. Load system prompt (CLAUDE.md)
    system_prompt = load_system_prompt()
    
    # 3. Load context files
    context = load_context_files()  # Business_Goals.md, Company_Handbook.md
    
    # 4. Build prompt for Claude
    prompt = f"""You are an AI Employee WORKER. Process the task and return ONLY JSON.

CRITICAL RULES:
1. Output ONLY JSON - no markdown, no text before or after
2. Do not include any explanations
3. Do not use code blocks or formatting
4. Just raw JSON

Task file: {task_file.name}

Task content:
{task_content}

=== SYSTEM PROMPT ===
{system_prompt}

=== BUSINESS GOALS ===
{context.get('business_goals', 'Not available')}

=== COMPANY HANDBOOK ===
{context.get('company_handbook', 'Not available')}

Output ONLY this JSON format (no other text):
{{
  "decision": "complete_task" | "create_approval_request" | "needs_revision",
  "category": "general" | "important" | "urgent" | "invoice" | "payment",
  "ai_action_taken": "What you did",
  "ai_response": "Your full response text"
}}

Examples:
Input: "Hey I am abdullah"
Output: {{"decision": "complete_task", "category": "general", "ai_action_taken": "Added greeting response", "ai_response": "Hello! Nice to meet you."}}

Output ONLY the JSON."""

    # 5. Invoke Claude Code
    result = invoke_claude(prompt, timeout=300)
    
    # 6. Parse and validate JSON
    decision = parse_and_validate_json(result.stdout)
    
    # 7. Fill template with Claude's response
    content = task_file.read_text()
    updated_content = fill_ai_placeholders(content, decision)
    
    # 8. Validate all [PENDING] replaced
    if not validate_no_pending_placeholders(updated_content):
        move_file(task_file, "Needs_Revision", "Placeholders not filled")
        return
    
    # 9. Write updated content
    task_file.write_text(updated_content)
    
    # 10. Execute decision
    move_based_on_decision(task_file, decision)
```

# Execute decision
decision_type = decision.get("decision")

if decision_type == "complete_task":
    # Move to Done/

## AI Analysis

{decision.get('ai_response')}

    approval_file = settings.vault_path / "Pending_Approval" / f"APPROVAL_{task_file.stem}.md"
    approval_file.write_text(approval_content)
    
    # Move task file to Pending_Approval/

elif decision_type == "needs_revision":
    # Move to Needs_Revision/

## 🎯 **Summary**

### **What We're Building:**

An **AI Employee** that:
- ✅ Watches multiple sources (files, emails, messages)
- ✅ Uses centralized templates for consistency
- ✅ Leverages Claude Code for analysis and decisions
- ✅ Follows strict JSON schema for responses
- ✅ Validates all responses before execution
- ✅ Executes decisions autonomously
- ✅ Logs everything for audit trail

### **How It Works:**

1. **Watchers** create metadata with `[PENDING]` placeholders
2. **Orchestrator** moves to Processing/ and calls Claude Runner
3. **Claude Runner** sends prompt to Claude Code with:
   - Task metadata + content
   - System prompt (CLAUDE.md)
   - Business goals
   - Company handbook
4. **Claude** returns strict JSON with decision
5. **Claude Runner** validates JSON, fills template, executes decision
6. **File** moved to appropriate folder (Done/, Pending_Approval/, Needs_Revision/)

### **Key Principles:**

- ✅ **Separation of Concerns:** Watchers watch, Claude thinks, Runner executes
- ✅ **Strict Structure:** JSON-only responses, validated schema
- ✅ **Template-Driven:** Centralized templates ensure consistency
- ✅ **Validation:** JSON schema + placeholder validation
- ✅ **Audit Trail:** Everything logged, nothing hidden
