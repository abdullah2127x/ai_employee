## 🔧 **Implementation Details**

### **BASE_METADATA_TEMPLATE**

Single template used by ALL watchers:

```python
BASE_METADATA_TEMPLATE = """---
type: {type}
task_id: {task_id}
original_name: {original_name}
{additional_yaml_fields}
detected: {detected}
priority: {priority}
status: pending
---

# {title}: {original_name}

{header_fields}

---

## Content

```
{content}
```

---

## Suggested Actions

{actions}

---

## Processing Notes

(Add notes here during processing)

---

*Generated by AI Employee {watcher_name}*
*Task ID: `{task_id}`*
"""
```

## 🎯 **Migration Guide**

### **For Existing Code:**

**Old:**
```python
metadata_content = self._prepare_metadata_content(
    source=source,
    content=content,
    size=size,
    file_hash=file_hash,
    priority=priority,
    timestamp=timestamp,
)
```

**New:**
```python
from utils.task_template import create_file_drop_metadata

metadata_content = create_file_drop_metadata(
    original_name=source.name,
    original_path=source,
    content=content,
    file_hash=file_hash,
    size=size,
    priority=priority,
    timestamp=timestamp,
)
```

### **For New Watchers:**

```python
from utils.task_template import create_email_metadata

metadata_content = create_email_metadata(
    from_address=email_from,
    subject=email_subject,
    content=email_body,
    timestamp=email_timestamp,
    priority=priority
)
```
